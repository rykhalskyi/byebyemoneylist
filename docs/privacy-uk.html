Політика конфіденційності для Bye-Bye Money List

Останнє оновлення: 10 червня 2026 р.

Ваша конфіденційність важлива для нас. Bye-Bye Money List розроблено так, щоб надати вам повний контроль над вашими даними. Ця Політика конфіденційності пояснює, як ми поводимося з вашою інформацією.

1. Збір даних

Bye-Bye Money List не збирає автоматично жодних персональних даних. Ми не відстежуємо ваше місцезнаходження, не отримуємо доступу до ваших контактів і не моніторимо ваші дії в додатку автоматично.

Усі дані в додатку вносяться вами вручну (наприклад, додавання продуктів, цін або категорій). Ці дані не є персоналізованими та використовуються виключно для функціонування додатка.

2. Зберігання даних

Усі дані, які ви вводите, зберігаються виключно на вашому пристрої. Ми не маємо центральних серверів для зберігання вашої історії покупок або списків. Ваші дані завжди залишаються під вашим фізичним контролем.

3. Поширення даних

Додаток не передає ваші дані третім особам, якщо ви явно не вирішите це зробити (наприклад, використовуючи функції експорту або поширення, передбачені в додатку).

4. LLM API та сканування чеків

Додаток містить необов'язкову функцію сканування чеків, яка використовує великі мовні моделі (LLM). Щоб скористатися цією функцією, ви повинні налаштувати постачальника LLM API на свій вибір (наприклад, OpenAI, Anthropic тощо), надавши власний ключ API.

Коли ви скануєте чек:

  • Зображення чека надсилається безпосередньо постачальнику API, якого ви налаштували.
  • Ми не зберігаємо це зображення. Зображення обробляється в режимі реального часу для вилучення даних і не зберігається додатком після завершення сканування.
  • Постачальники API є сторонніми сервісами. Ми не несемо відповідальності за їхні методи обробки даних або політику конфіденційності. Ми рекомендуємо вам ознайомитися з політикою конфіденційності того постачальника, якого ви вирішите використовувати.

5. Видалення даних

Оскільки всі дані зберігаються локально на вашому пристрої, ви можете видалити їх у будь-який час:

  • Скориставшись опцією «Очистити дані» в налаштуваннях системи Android для цього додатка.
  • Видаливши додаток зі свого пристрою.

Після видалення ці дані не можуть бути відновлені нами, оскільки ми не маємо жодних резервних копій на наших серверах.
